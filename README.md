

EJERCICIO PRÁCTICO - PORTAFOLIO MÓDULO 5
Aplicación Bursatil

Este proyecto es una aplicación web de consulta bursátil desarrollada como parte del portafolio profesional. Permite a los usuarios visualizar indicadores financieros reales.

Características

- Interfaz de registro/ingreso.
- visualización de datos en tarjetas.
- Manejo de sesiones y almacenamiento de historial (LocalStorage).
- Implementación de ES6
- Consulta de datos mediante API externa (mindicador.cl).

Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript ES6
- LocalStorage
